# Chronos

## Motivation
Project Laboratoria Hackaton 12hs - Create a web-app that uses an Open Weather API to create scripts and an Open Weather API to inform the weather of the city.

## Technology used
* HTML
* CSS
* Bootstrap
* Javascript
* Jquery
* Firebase
* API Here Maps
* API Open Weather
